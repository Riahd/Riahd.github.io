# Portfolio Rïad Ouabdelkader

Portfolio personnel de Rïad Ouabdelkader, étudiant en BTS SIO option SISR.

## Description

Ce site web one-page présente mon parcours, mes compétences, mes projets et mes expériences professionnelles dans le domaine de l'informatique, spécialisation systèmes et réseaux.

## Structure du site

- **Accueil** : Présentation générale avec aperçu des projets
- **À propos** : Informations personnelles et compétences
- **Parcours** : Formation et expériences
- **Projets** : Réalisations techniques
- **Expériences** : Stages et emplois
- **Contact** : Coordonnées

## Technologies utilisées

- HTML5
- CSS3 (thème sombre, animations, responsive design)
- JavaScript (animations de particules avec connexions)

## Fonctionnalités

- Thème sombre moderne avec effets de lumière et ombres
- Animations d'entrée pour les éléments
- Système de particules interactives avec connexions
- Design responsive pour mobile et desktop
- Navigation fluide avec ancres

## Comment lancer le site

Ouvrez simplement le fichier `index.html` dans votre navigateur web pour commencer.

## Personnalisation

Pour modifier le contenu, éditez les sections dans `index.html`. Les styles sont intégrés pour un thème cohérent.

### Ajouter une photo de profil

Remplacez `profile-pic.png` par votre photo (format PNG recommandé, taille 150x150px).

## Inspirations

Style inspiré de portfolios modernes avec thème sombre et animations de particules.
