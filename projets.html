Projet 1 — Mise en place d'un environnement Windows Server

🎓 Projet scolaire — BTS SIO SISR

🎯 Objectif

Créer un domaine Active Directory avec gestion des utilisateurs, droits d'accès, intégration de postes clients et configuration DNS.

Windows Server → AD DS → DNS → Clients

⚙️ Ce que j'ai réalisé

  • J'ai intégré un poste client au domaine et configuré le DNS

Projet 2 — Centralisation des logs (Promtail / Grafana / Loki)

🎓 Projet scolaire — BTS SIO SISR

🎯 Objectif

Mettre en place une stack Promtail + Loki + Grafana pour collecter, centraliser et visualiser les logs de plusieurs machines via des dashboards.

Promtail → Loki → Grafana

⚙️ Ce que j'ai réalisé

  • J'ai installé Loki comme serveur de stockage de logs
  • J'ai déployé Promtail sur les machines pour la collecte
  • J'ai configuré Grafana et créé des dashboards de supervision
  • J'ai testé la remontée des logs et vérifié la cohérence des données

🖼️ Captures

Proxmox — serveur hébergeant la collecte de logs

Grafana — visualisation et filtrage des logs via Loki
